# contact-app

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Json server installation
```
npm i json-server
```

### Run the server
```
go to folder DB
json-server --watch db.json
```
